Variables

¿Qué es una variable?
Una variable es un espacio en memoria que permite almacenar información para utilizarla posteriormente en un programa.

Ejemplo en JavaScript
let nombre = "Yeraldyn";
let edad = 20;
let estudiante = true;
Tipos de datos básicos
String
Almacena texto.

let nombre = "Yeraldyn";
Number
Almacena números.

let edad = 20;
Boolean
Almacena valores verdadero o falso.

let estudiante = true;

Conclusión

Las variables son fundamentales en programación porque permiten almacenar y manipular información durante la ejecución de un programa.
