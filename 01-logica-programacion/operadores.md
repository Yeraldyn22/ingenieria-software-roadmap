# Operadores

## ¿Qué son los operadores?

Los operadores permiten realizar operaciones sobre variables y valores.

## Operadores Aritméticos

```javascript
let suma = 10 + 5;
let resta = 10 - 5;
let multiplicacion = 10 * 5;
let division = 10 / 5;
```

## Operadores de Comparación

```javascript
10 == 10
10 != 5
10 > 5
10 < 20
```

## Operadores Lógicos

```javascript
true && true
true || false
!true
```

## Ejercicio

Crear variables numéricas y realizar:

* Suma
* Resta
* Multiplicación
* División

## Conclusión

Los operadores permiten manipular datos y tomar decisiones dentro de un programa.
